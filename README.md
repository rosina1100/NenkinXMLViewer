# NenkinXMLViewer

日本年金機構から届く XML 形式の公文書を、macOS でワンクリックで閲覧できるネイティブアプリです。

## なぜこのアプリが必要か

日本年金機構の電子送付（e-Gov 電子申請）で届く公文書は XML 形式です。Windows では IE モード廃止以降「XML が開けない」問題が繰り返し発生し、有料ビューアも存在するほどの需要があります。

macOS の Safari は XSL 変換に対応しているため表示自体は可能ですが、以下の不満がありました：

- **印刷・PDF 化が崩れる** — 年金機構の XSL は画面表示のみ想定で、印刷レイアウトが考慮されていない
- **専用アプリとして使えない** — Safari で開くと他のタブに紛れる

NenkinXMLViewer はこれらを解決する **macOS 向けの年金機構 XML 専用ビューア**です。

## 機能

### 2つの表示モード

- **XSL 変換表示** — 同フォルダの XSL を自動検出し、年金機構の公式レイアウトで表示
- **ビルトイン表示** — 印刷・PDF 化に最適化された独自レイアウトで表示

ワンクリックで両モードを切り替えられます。

### 対応する文書タイプ（ビルトイン表示）

- 社会保険料額情報
- 基本保険料算出内訳書
- 保険料増減内訳書
- 標準報酬決定通知書
- CSV形式届書総括票
- 鏡文書（送付状）
- その他の XML（XSL 変換で表示）

### その他の機能

- **右クリック → 「このアプリケーションで開く」** でXMLファイルから直接起動
- **ドラッグ＆ドロップ** — ウィンドウに XML をドロップして表示
- **タブ表示** — 鏡文書内の添付ファイルリンクをクリックすると新しいタブで開く
- **CSV 表示** — 添付の CSV ファイルもテーブル形式で閲覧可能
- **PDF 表示** — 添付の PDF もアプリ内で閲覧可能（保存ボタン付き）
- **印刷 / PDF 保存** — ⌘P で A4 に最適化された印刷ダイアログを表示
- **エンコーディング自動判定** — UTF-8 / Shift_JIS / EUC-JP に対応

## 動作環境

- macOS 14 (Sonoma) 以降
- Xcode 15 以降（ビルド時のみ）

## インストール

### ビルド済みアプリを使う場合

[Releases](../../releases) からダウンロードし、`/Applications/` に配置してください。

### 自分でビルドする場合

```bash
git clone https://github.com/rosina1100/NenkinXMLViewer.git
cd NenkinXMLViewer
open NenkinXMLViewer.xcodeproj
```

Xcode で ⌘R（ビルド＆実行）するだけです。

## 使い方

### 初回設定

1. Finder で年金機構の `.xml` ファイルを右クリック
2. 「情報を見る」(⌘I) → 「このアプリケーションで開く」で NenkinXMLViewer を選択
3. 「すべてを変更...」をクリック

以降、XML ファイルをダブルクリックするだけで NenkinXMLViewer が起動します。

### 表示モードの切り替え

XML を開くと、XSL ファイルが同フォルダにあれば XSL 変換表示、なければビルトイン表示で開きます。画面上部の切り替えボタンでいつでもモードを変更できます。

印刷や PDF 保存をしたい場合は、**ビルトイン表示**に切り替えてから ⌘P を押すことをおすすめします。

## アーキテクチャ

```
┌─────────────────────────────────────────┐
│              SwiftUI App                │
│  ┌──────────┐  ┌─────────────────────┐  │
│  │AppDelegate│  │   ContentView       │  │
│  │ファイル受渡│  │ ┌─────────────────┐ │  │
│  └──────────┘  │ │  WKWebView      │ │  │
│                │ │  ┌───────────┐  │ │  │
│   Swift ←──────┤ │  │viewer.html│  │ │  │
│   JS通信       │ │  │ XSLT変換  │  │ │  │
│   (Base64)     │ │  │ ビルトイン │  │ │  │
│   ──────→ JS   │ │  └───────────┘  │ │  │
│                │ └─────────────────┘ │  │
│                └─────────────────────┘  │
└─────────────────────────────────────────┘
```

- **Swift 側**: ファイルの読み込み、同フォルダの XSL 自動検出、エンコーディング判定、サンドボックス対応
- **JavaScript 側**: XSLT 変換、ビルトイン表示のレンダリング、表示モード切替
- **通信**: Swift → JS は Base64 エンコード、JS → Swift は `WKScriptMessageHandler`
